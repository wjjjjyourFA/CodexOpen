## **GPS 和 GNSS 的区别主要在于范围和归属：**

### 1. 定义不同

- **GPS (Global Positioning System)**
  - 全称是 **全球定位系统**。
  - 是 **美国**研发并维护的 **卫星导航系统**。
  - 目前是最早投入使用、最广泛应用的卫星导航系统。
- **GNSS (Global Navigation Satellite System)**
  - 全称是 **全球导航卫星系统**。
  - 是一个总称，指代 **所有国家和地区的卫星导航系统** 的集合。
  - GPS 只是 GNSS 的一部分。

------

### 2. 组成系统

GNSS 包括以下主要系统：

- **GPS**：美国的全球定位系统。
- **GLONASS**：俄罗斯的全球卫星导航系统。
- **Galileo**：欧盟的伽利略系统。
- **北斗 (BeiDou, BDS)**：中国的北斗导航系统。
- 以及日本的 **QZSS**、印度的 **NavIC** 等区域导航系统。

------

### 3. 区别总结

- **GPS 是一个具体的系统**，GNSS 是一个大类。
- **GPS ⊂ GNSS**，即 GPS 属于 GNSS。
- 用通俗比喻：
  - **GNSS 就像“水果”**，
  - **GPS 就像“苹果”**。
